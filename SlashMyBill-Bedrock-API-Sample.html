2. Python SDK Call

# boto3 Bedrock Runtime invoke_model call
import boto3, json

bedrock = boto3.client('bedrock-runtime', region_name='us-east-1')

request_body = {
    "messages": [
        {
            "role": "user",
            "content": [{"text": prompt}]
        }
    ],
    "inferenceConfig": {
        "max_new_tokens": 5000,
        "temperature": 0.3,
        "top_p": 0.9
    }
}

response = bedrock.invoke_model(
    modelId='amazon.nova-lite-v1:0',
    contentType='application/json',
    accept='application/json',
    body=json.dumps(request_body)
)

result = json.loads(response['body'].read())
text = result['output']['message']['content'][0]['text']

5. Data Flow Summary

Bill PDF → pdfplumber → Parsed Bill Data
                              ↓
DynamoDB Tips ──────→ Build Prompt (bill + tips + rules)
                              ↓
                    Bedrock invoke_model
                    (Nova Lite, 5000 tokens)
                              ↓
                    JSON Response (summary,
                    service_analysis[], savings_plan_analysis)
                              ↓
                    PDF Generator (ReportLab + PyPDF2)
                    → Monthly/Yearly Min/Max/Avg savings
                    → Per-service savings breakdown
                    → Comma-formatted costs ($1,234.56)
                              ↓
                    Merged PDF Report → S3 → Pre-signed URL

6. Error Handling

ErrorCodeAction
ThrottlingException429Retry with exponential backoff (1s, 2s)
ServiceUnavailableException503Retry with exponential backoff (1s, 2s)
Invalid JSON response—Raise ValueError, fail the analysis
Missing required fields—Raise ValueError with field name
All retries exhausted—Raise RuntimeError, return error to user
