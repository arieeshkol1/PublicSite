Copy ▶ Paste Wizard

Step 1 — Select direction

Choosing Input will auto-target Output (and vice‑versa).

Step 2 — Browse Input like Explorer

Endpoints → Folders: | Files:
The UI calls /list-(input|output)?type=folder|file&prefix=.... Double‑click a folder to navigate; click a file to select it for copy.

Step 3 — Execute

POST to /copy with payload: { fromBucket, toBucket, mode, key }
