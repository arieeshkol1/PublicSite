Step 2: Deploy the Template in AWS

Open the AWS CloudFormation console and create a new stack using the template you just downloaded.

  1. Sign in to the AWS CloudFormation Console with the account 123456789012
  2. Click "Create stack" → "With new resources (standard)"
  3. Select "Upload a template file" and choose the file you downloaded
  4. Click "Next", enter a stack name (e.g., SlashMyBill-Access)
  5. Click "Next" through the options page
  6. On the review page, check "I acknowledge that AWS CloudFormation might create IAM resources with custom names"
  7. Click "Submit" and wait for the stack to show CREATE_COMPLETE
